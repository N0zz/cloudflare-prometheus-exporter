# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do not** open a public GitHub issue
2. Use [GitHub Security Advisories](https://github.com/N0zz/cloudflare-prometheus-exporter/security/advisories/new) to report the vulnerability privately
3. Include steps to reproduce the issue and any relevant details

You should receive a response within 7 days.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |

## Security Scanning

This project uses automated security scanning:

- **Trivy** — container and dependency vulnerability scanning
- **CodeQL** — static application security testing
- **Bandit** — Python-specific security analysis
- **OpenSSF Scorecard** — supply chain security assessment

Results are available in the [GitHub Security tab](https://github.com/N0zz/cloudflare-prometheus-exporter/security).
